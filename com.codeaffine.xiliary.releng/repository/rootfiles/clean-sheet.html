Clean Sheet

An ergonomic theme for the Eclipse Java IDE on Windows 10
Attuned to lessen visual fatigue and eyestrain
With a clean and glare free look and feel
Based on decent color contrasts
Focused on readability

Theme Selection

Preferences: General > Appearance > Theme: Clean Sheet

Dark Theme with FlatScrollbars

For fans of the windows dark theme, the Code Affine Theme feature provides a derivation that incorporates the Xiliary FlatScrollBar component on trees and tables.

Theme: Dark [Flat-Scrollbar]

Default Windows Theme with FlatScrollbars

For fans of the default windows theme, the Code Affine Theme feature provides a derivation that incorporates the Xiliary FlatScrollBar component on trees and tables.

Theme: Windows [Flat-Scrollbar]

Note

The FlatScrollbar overlay mechanism is currently only available for table and tree based UI components. Due to the variety of the Eclipse ecosystem it might not work properly in all scenarios. Issue and usability reports to improve the user experience are highly appreciated.
